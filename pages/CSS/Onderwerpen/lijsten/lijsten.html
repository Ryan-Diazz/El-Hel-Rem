Lijsten

In HTML kan je lijsten maken om een duidelijk overzicht te maken van onderwerpen, ingredienten of informatie.
deze lijsten kan je met CSS aanpassen om het mooier te maken met de volgende CSS eigenschap.

  • list-style-type, met deze eigenschap kan je aanpassen hoe de lijst gemarkeerd wordt.
  • list-style-image, met deze eigenschap kan je een foto als markering gebruiken.
  • list-style, deze eigenschap kan je gebruiken om de waardes van de bovenste eigenschappen in één eigenschap te zetten.

ul lijsten, list-style-type

ul lijsten zijn ongeordend, dus deze lijsten hebben geen vaste volgorde en is meer bedoeld om op een overzichtelijke manier een lijst te geven van verschillende onderwerpen.
maar bij HTML zijn de markeringen automatisch bolletjes maar die kan je ook veranderen naar andere vormpjes met de volgende waardes voor de css eigenschap list-style-type:

  • circle
  • disk
  • square

ol, list-style-type

bij geordende lijsten zijn de markeringen genummerd en op volgorde. deze nummering is nu decimaal, dus gewoon 1, 2, 3, 4, 5 ... enz.
deze nummering kan je veranderen met de list-style-type eigenschap in css.
je hebt vele waardes voor deze eigenschap die je kan gebruiken om de nummering te veranderen naar een nummering die in de context wat toepasselijker is.

je hebt de volgende waardes voor deze eigenschap:

  • decimal
  • decimal-leading-zero
  • lower-roman
  • upper-roman
  • lower-alpha
  • upper-alpha
  • lower-greek
  • upper-greek
  • lower-latin
  • upper-latin
  • hebrew
