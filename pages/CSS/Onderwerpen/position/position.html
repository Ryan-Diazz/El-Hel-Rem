Position

Je kan elementen in HTML verschillende posities geven met de position eigenschap in CSS.
met deze eigenschap ken je elementen verschillend laten reageren op scrollen de manier dat het element wordt weergegeven.

Bij de position eigenschap kan je de volgende waardes gebruiken:

  • static (standaard)
  • relative
  • fixed
  • absolute
  • sticky

position: static;

Elementen met de CSS eigenschap static gaan mee met de rest van de pagina, dus ook als er wordt gescrolled.
elementen met deze waarde kun je niet verplaatsen met de top, bottom, right en left CSS eigenschappen.

position: relative;

Elementen met de CSS eigenschap relative kan je wel verplaatsen met de top, bottom, right en left CSS eigenschappen.
Als een element deze waarde heeft dan gaan de elementen niet de open ruimtes opvullen om het element heen
de positie van het element is dan ook relatief aan waar het element normaal zit.

CSS code:

position: fixed;

Als een element deze eigenschap heeft dan blijft het element op dezelfde plek in het beeld als waar het hoorde, het verplaatst zich niet als er wordt gescrollt en blijft altijd op dezelfde positie.
deze elementen kunnen worden verplaatst met de top, bottom, right en left CSS eigenschappen.

position: absolute;

Position absolute doet hetzelfde als position: fixed; maar inplaats dat het element in het beeld altijd op dezelfde plek blijft, blijft het element altijd op dezelfde plek ten opzichte van het bovenliggende element.
het bovenliggende element moet echter wel een position hebben ander wordt de position absolute element opdezelfde plek gehouden ten opzichten van de body.
Elementen met deze waarde kun je verplaatsen met de top, bottom, right en left CSS eigenschappen.

CSS code:

position: sticky;

Met deze eigenschap wisselt eigenlijk tussen Position: relative; en position: fixed;, Als een element deze eigenschap heeft dan heeft het element eigenlijk position: relative;,
maar als een bepaalde positie is berijkt dan verandert de eigenschap naar position: fixed; en blijft hij op die plek plakken.
Bij dit element moet je één van de top. bottom, right en left eigenschappen gebruiken om de waarde te defineren waar het element moet blijven plakken.
