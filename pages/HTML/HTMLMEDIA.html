Afbeeldingen op je pagina zetten

Als je een afbeelding op je website wilt zetten dat gebruik je het img-element. Het is natuurlijk niet zo dat je alleen img erin zet, je moet er een hele code bij schrijven.
Er horen nog allemaal verschillende codes bij, namelijk: de code begint met src , src is afkomstig van source. In de code source schrijf je de url van de afbeelding.
Met url wordt bedoeld de bestands naam van de afbeelding die bij zo'n soort code moet de afbeelding worden opgeslagen en die moet dan eindigen op -.png of -.jpg anders kan de
site jouw afbeedling niet vinden. Jouw website gaat er ook vanuit dat de afbeelding in hetzelfde document als het html bestand staat als dat niet zo is moet je dat
aangeven dat doe je door ../mapnaam/fotnaam erbij te zetten.

Vervolgens kom alt , alt zorgt ervoor dat als jouw afbeelding niet kan laden op de internet pagina, bijvoorbeeld door slecht wifi
dat er een alternatieve tekst komt te staan, zodat mensen snappen wat voor een afbeelding er staat.
Je moet een alt-attribuut in je code zetten, hij mag ook leeg zijn, maar als je het niet doet dan komt er een foutmelding.

Vervolgens komt de titel , een titel is niet verplicht, maar het is wel leuk om het in te stellen.
Als je een titel in je code zet dan wordt er letterlijk een titel gegeven aan je afbeelding, deze verschijnt alleen als je met je muis op de afbeelding gaat staan
of er langs gaat.

Het laatste stukje code gaat over height en width , dit gebruik je voor de grootte van de foto. Hoe hoog en breed jij je afbeelding wilt hebben.
Dit gaat altijd in pixels. Als je de verkeerde breedte en hoogte opgeeft krijgt je afbeelding een hele rare vorm en kan de pagina het niet goed laden en krijgt het dus een hele rare vorm.
